製品名
Window Thumb Wall

概要
複数のウィンドウを同時に見渡せる、低負荷なライブサムネイル壁面アプリです。

詳細説明
Window Thumb Wall は、開いている複数のウィンドウをライブサムネイルとして並べ、見やすい壁面レイアウトで表示できる Windows デスクトップアプリです。

アプリを何度も切り替えなくても、必要なウィンドウを一度に見続けられます。ビルドログ、ダッシュボード、チャット、ブラウザ、端末、チャート、監視画面など、作業や確認に必要な情報を 1 つの画面にまとめて表示できます。

表示には Windows 標準の Desktop Window Manager サムネイル API を使っているため、画面キャプチャや配信ソフトのような重い構成なしで、軽快にライブ表示を維持できます。

このアプリが向いている用途:
- 複数のウィンドウを同時に監視したい。
- Alt+Tab の切り替え回数を減らしたい。
- 余っている画面領域を監視壁面として活用したい。
- 静止画ではなくライブの更新状態をそのまま見たい。
- 点滅中のウィンドウを赤枠で見つけやすくしたい。
- サムネイルから元ウィンドウへすぐ戻りたい。

主な機能
- 外部ウィンドウのライブサムネイル表示。
- 複数ウィンドウに対応した柔軟な壁面レイアウト。
- 対象ウィンドウを素早く追加できるウィンドウ選択リスト。
- サムネイルクリックで元ウィンドウをアクティブ化。
- タスクバー点滅に連動した赤枠通知。
- 監視に向いたフルスクリーン表示。
- 最前面表示。
- テレメトリなしのローカル完結動作。

注意
本アプリは現状有姿で提供されます。Windows のバージョン、モニタ構成、監視対象アプリによって動作や相性が異なる場合があります。重要な用途に利用する前に、ご自身の環境で事前にご確認ください。
